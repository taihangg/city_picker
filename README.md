做天气预报的功能时，需要获取城市code，用typeahead做了个选择城市的小插件，可以点选，也可以搜索。勉强能用。

需要自己下载typeahead包
(https://github.com/AbdulRahmanAlHamali/flutter_typeahead)
(https://pub.dev/packages/flutter_typeahead)
